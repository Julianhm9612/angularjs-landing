1. Install workflow yeoman

    > npm install -g yo

2. Install scaffolding

    > npm install -g generator-angular

3. Crete the app folder

    > mkdir angularjs-landing

4. Install dev dependencies

    > npm install

5. Install bower

    > npm install -g bower

6. Install angular js and his modules

    > bower install

7. Serve

    > grunt serve --force
